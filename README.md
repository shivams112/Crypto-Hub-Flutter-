# About Crypto Hub

A beautiful and smooth application totally made with Flutter through which a user can track the current scenario i.e price and hikes of various Crypto Currencies in the market. The reason why I made this application is to brush up my skills in APIs handling and Asynchronous tasks in Flutter.

PS-> There are many features which I'd definitely try to implement in future. 

## Installation


```bash
flutter build apk
```
```bash
flutter install
```
